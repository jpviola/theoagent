# Contribuyendo a SantaPalabra

¡Gracias por tu interés en contribuir a SantaPalabra! Este proyecto es de código abierto y agradecemos la colaboración de la comunidad para crear la mejor herramienta de asistencia teológica católica.

## Cómo contribuir

1.  **Haz un Fork** del repositorio.
2.  **Crea una rama (branch)** para tu funcionalidad o corrección (`git checkout -b feature/nueva-funcionalidad`).
3.  **Realiza tus cambios** (asegúrate de seguir el estilo de código existente).
4.  **Haz Commit** de tus cambios (`git commit -m 'Añadir nueva funcionalidad'`).
5.  **Haz Push** a la rama (`git push origin feature/nueva-funcionalidad`).
6.  **Abre un Pull Request**.

## Reportar errores

Si encuentras un error, por favor abre un "Issue" describiendo el problema, cómo reproducirlo y qué comportamiento esperabas.

## Estándares de Código

*   Usamos **TypeScript** y **Next.js**.
*   Mantén los componentes pequeños y reutilizables.
*   Usa Tailwind CSS para los estilos.
*   Asegúrate de que el código sea legible y esté bien comentado cuando sea necesario.
